# TecnoTV
Handle Channel lists of m3u8 extracted from TecnoTV

# Description
Sends channel to my Roku device. 
# HOW-TO
Click on the jLabel with the representative image and enjoy.
# Achievements
* It even works on Android thanks to JRE4Android
